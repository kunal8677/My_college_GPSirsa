COMPUTER ENGG.

Computer Engineering, at the diploma level, is a dynamic and rewarding field that provides a strong foundation in the principles and practices of computer systems and technology. This program offers a unique opportunity for individuals to enter the world of technology and engineering with a practical, hands-on approach.

Course Duration - 3 years
Degree Level - Diploma
Semester Fees - 2850/-

1st Semester

Subjects :-

  • English and Communication Skills - I
  • Applied Mathematics - I
  • Applied Physics - I
  • Fundamentals of IT
  • Computer Workshop
  • Electronics Workshop

2nd Semester

Subjects :-

  • Advances in IT
  • Applied Mathematics-II
  • Applied Physics - II
  • Analog Electronics
  • Engineering Graphics
  • Multimedia Applications
  • Environmental Studies & Disaster Management

3rd Semester

Subjects :-

  • Industrial/In-House Training - I
  • Operating Systems
  • Digital Electronics
  • Programming in C
  • Data Base Management System

4th Semester

Subjects :-

  • English and Communication Skills - II
  • Computer Organisation & Architecture
  • Data Structures using C
  • Object Oriented Programming using Java
  • Open Elective (MOOCs+ /Offline)
  • Minor Project

5th Semester

Subjects :-

  • Software Engineering
  • Computer Networks
  • Computer Programming Using Python
  • Cloud Computing
  • Web Development Using PHP

6th Semester

Subjects :-

  • Network Security
  • Mobile Application Development
  • Application Development Using Web Framework
  • Entrepreneurship Development & Management
  • Project Work

News Letters

Govt. Polytechnic is situated in the west of the Sirsa City. Its distance is about 5 KM from Main Bus stand & about 4 KM from the Railway station of the city on Rania/Ellenabad road. The institution is spread over an area of 26 acres and has a very good & green campus. It came into existence in 1962 at the time of Joint Punjab by the efforts of Sh. Sardar Partap Singh Kairon, the then Chief Minister of the Joint Punjab.
